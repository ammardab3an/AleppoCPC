      Hey, my approach for Segment with the Required Subset was: I used the two stacks trick that was in the tutorials. Now we only have to recalculate the subset sum when we add an element. We can store the intermediate DP-table of a subset sum as a bitset of size 1001, where b[i] means that you can reach sum i. Then if you add an element to the set, the bitset gets updated like: b = (b | b<<val). . Last thing left is to check if current segment is good. We have to merge the top bitset of the first and second stack. This can be done with the bitset and-operation. Only then you have to store one of the bitsets backwards, so the elements line up. The runtime will be

      O(n⋅smaxwordsize)O(n⋅smaxwordsize)

      In the first problem you can't take the same element twice. The playlist will always go to the next song.

Could someone help me with Problem F of Step 2 (I fail on test case 3)? My idea is to maintain an increasing and decreasing sequence of numbers which will allow me to find updates quickly. When I reach a new maximum, I update the minimum so that (max — min) becomes at most k. Similarly for the case when I reach a new minimum. Any help would be really appreciated, thanks!

Code
#include <bits/stdc++.h>

constexpr bool test_cases = false;

void solve () {
    int64_t n, k;
    std::cin >> n >> k;
    std::vector <int64_t> a (n);
    for (int i = 0; i < n; ++i)
        std::cin >> a [i];

    int index = 0;
    int min   = a [index];
    int max   = a [index];
    std::deque <int> asc;
    std::deque <int> dsc;
    int64_t ans = 1;

    asc.push_back(0);
    dsc.push_back(0);

    for (int i = 1; i < n; ++i) {
        while (!asc.empty() && a [asc.back()] > a [i])
            asc.pop_back();
        while (!dsc.empty() && a [dsc.back()] < a [i])
            dsc.pop_back();

        asc.push_back(i);
        dsc.push_back(i);

        if (a [i] > max) {
            max = a [i];
            if (max - min > k) {
                while (max - a [asc.front()] > k)
                    asc.pop_front();
                index = asc.front();
                min   = a [index];
                while (dsc.front() < index)
                    dsc.pop_front();
            }
        }
        else if (a [i] < min) {
            min = a [i];
            if (max - min > k) {
                while (a [dsc.front()] - min > k)
                    dsc.pop_front();
                index = dsc.front();
                max   = a [index];
                while (a [asc.front()] < index)
                    asc.pop_front();
            }
        }

        ans += i - index + 1;
    }

    std::cout << ans << '\n';
}

int main () {

    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int32_t cases = 1;
    if (test_cases)
        std::cin >> cases;
    while (cases--)
        solve();

    return 0;
}

            while (currSum >= sum) {
                currSum = currSum - arr[l1];
                ++l1;
            }
            ans = Math.max(ans, i - l1 + 1);
        }
        bw.write(ans + "\n");
    }
    sc.close();
    bw.flush();
}

[SPOILER ALERT] Easy AC solution for the problem "Segment with the Required Subset"

While iterate over the input array, you should maintain an array C of length s where: C[j] is the maximum index of the beginning subsequence whose sum is equal to j.

k = 0 # this is the second pointer
C = [-1 for i in range(s)]
ans = int('inf')
# Iterate the array with the first pointer
for i in range(0, n, 1):
    # Update C array
    for j in range(s, a[i], -1):
        C[j] = max(C[j], C[j-a[i]])
    C[a[i]] = i
  
    if C[s] >= k:
        ans = min(ans, i - C[s] + 1)
        k = C[s] + 1

print(ans if ans < int('inf') else -1)
