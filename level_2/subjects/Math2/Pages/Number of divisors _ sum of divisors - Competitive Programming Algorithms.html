    In this article we discuss how to compute the number of divisors d(n)d(n) and the sum of divisors σ(n)σ(n) of a given number nn.

    Number of divisors

    It should be obvious that the prime factorization of a divisor dd has to be a subset of the prime factorization of nn, e.g. 6=2⋅36=2⋅3 is a divisor of 60=22⋅3⋅560=22⋅3⋅5. So we only need to find all different subsets of the prime factorization of nn.

    Usually the number of subsets is 2x2x for a set with xx elements. However this is no longer true, if there are repeated elements in the set. In our case some prime factors may appear multiple times in the prime factorization of nn.

    If a prime factor pp appears ee times in the prime factorization of nn, then we can use the factor pp up to ee times in the subset. Which means we have e+1e+1 choices.

    Therefore if the prime factorization of nn is pe11⋅pe22⋯pekkp1e1⋅p2e2⋯pkek, where pipi are distinct prime numbers, then the number of divisors is:

    d(n)=(e1+1)⋅(e2+1)⋯(ek+1)d(n)=(e1+1)⋅(e2+1)⋯(ek+1)

    A way of thinking about it is the following:

    • If there is only one distinct prime divisor n=pe11n=p1e1, then there are obviously e1+1e1+1 divisors (1,p1,p21,…,pe111,p1,p12,…,p1e1).

    • If there are two distinct prime divisors n=pe11⋅pe22n=p1e1⋅p2e2, then you can arrange all divisors in form of a tabular.

      1p1p21⋮pe1111p1p21⋮pe11p2p2p1⋅p2p21⋅p2⋮pe11⋅p2p22p22p1⋅p22p21⋅p22⋮pe11⋅p22…………⋱…pe22pe22p1⋅pe22p21⋅pe22⋮pe11⋅pe221p2p22…p2e211p2p22…p2e2p1p1p1⋅p2p1⋅p22…p1⋅p2e2p12p12p12⋅p2p12⋅p22…p12⋅p2e2⋮⋮⋮⋮⋱⋮p1e1p1e1p1e1⋅p2p1e1⋅p22…p1e1⋅p2e2
      So the number of divisors is trivially (e1+1)⋅(e2+1)(e1+1)⋅(e2+1).

    • A similar argument can be made if there are more then two distinct prime factors.

    Sum of divisors

    We can use the same argument of the previous section.

    • If there is only one distinct prime divisor n=pe11n=p1e1, then the sum is:

      1+p1+p21+⋯+pe11=pe1+11−1p1−11+p1+p12+⋯+p1e1=p1e1+1−1p1−1

    • If there are two distinct prime divisors n=pe11⋅pe22n=p1e1⋅p2e2, then we can make the same table as before. The only difference is that now we now want to compute the sum instead of counting the elements. It is easy to see, that the sum of each combination can be expressed as:

      (1+p1+p21+⋯+pe11)⋅(1+p2+p22+⋯+pe22)(1+p1+p12+⋯+p1e1)⋅(1+p2+p22+⋯+p2e2)
      =pe1+11−1p1−1⋅pe2+12−1p2−1=p1e1+1−1p1−1⋅p2e2+1−1p2−1

    • In general, for n=pe11⋅pe22⋯pekkn=p1e1⋅p2e2⋯pkek we receive the formula:

      σ(n)=pe1+11−1p1−1⋅pe2+12−1p2−1⋯pek+1k−1pk−1σ(n)=p1e1+1−1p1−1⋅p2e2+1−1p2−1⋯pkek+1−1pk−1

    Multiplicative functions

    A multiplicative function is a function f(x)f(x) which satisfies

    f(a⋅b)=f(a)⋅f(b)f(a⋅b)=f(a)⋅f(b)
    if aa and bb are coprime.

    Both d(n)d(n) and σ(n)σ(n) are multiplicative functions.

    Multiplicative functions have a huge variety of interesting properties, which can be very useful in number theory problems. For instance the Dirichlet convolution of two multiplicative functions is also multiplicative.
